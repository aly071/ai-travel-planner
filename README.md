# AI Travel Planner

## Project info

AI Travel Planner is an AI-powered travel planning app that generates personalized day-by-day itineraries based on your destination, budget, dates, and interests.

**URL**: (add your Vercel URL after deployment)

## Features

- AI-generated itineraries using Claude API
- Interactive map view with activity pins
- Save and manage trips
- Export itinerary to PDF
- User authentication
- 6 fully designed screens: Welcome, Plan, Itinerary, My Trips, Saved, and Profile

## What technologies I used for this project?

This project is built with:

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL (Neon)
- Clerk (Authentication)
- Anthropic Claude API
- Leaflet (Maps)
